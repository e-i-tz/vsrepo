Установчный bash-скрипты для выполнения лабораторный работ по курсы **Безопасность вычислительных систем**. 

### Порядок действий:

1) Загрузить файлы на машину:

    1)  Клонировать репозиторий: 
    ``` bash

    sudo git clone https://github.com/e-i-tz/vsrepo.git
    ```
    2) Скачать zip-архив на машину (Code -> Download zip):

    <p align="center"> <img width="431" height="383" alt="image" src="https://github.com/user-attachments/assets/bb8c9c8b-a643-4f44-b475-e972505a4f49" />
    
2) Выдать права на исполнение .sh-файлам:
    ``` bash
    chmod +x ntopng.sh suricata.sh suricataips.sh grafana.sh 
    ```
3) Запуск (Пример):
    ``` bash
    sudo .\ntopng.sh  
    ```
